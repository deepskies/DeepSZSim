   Parameters:
   -----------
   image - float array
       the graph we are plotting
   title - str
       title of the graph
   xlabel - str
       label of the x-axis
   ylabel - str
       label of the y-axis
   cbarlabel - str
       label of the color bar
   width: int
       half rounded down of the width of output plot in pixels (eg, image size = 2*width+1)

   Returns:
   -------
   none
